# § Details

It would be better for you to understand the structure of this Datatable before the details.

The source tree [`src/`](https://github.com/OneWayTech/vue2-datatable/tree/master/src) is shown as below:

```
 ├─ _mixins/
 ├─ _utils/
 ├─ HeaderSettings/
 │   ├─ ColumnGroup.vue
 │   └─ index.vue
 ├─ Table/
 │   ├─ HeadSort.vue
 │   ├─ index.vue
 │   ├─ MultiSelect.vue
 │   ├─ TableBody.vue
 │   ├─ TableFooter.vue
 │   ├─ TableFrame.vue
 │   └─ TableHeader.vue
 ├─ index.vue
 ├─ main.js
 ├─ PageSizeSelect.vue
 └─ Pagination.vue
```
